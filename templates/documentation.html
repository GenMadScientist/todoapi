DOCUMENTATION FOR THE TODO API

Todo app urls

  1. path('', views.index)

    This takes the user to the todo list page

  2. path('todo_details/', views.todo_details,name="todo_details"),

    This takes the user to the details page of a particular todo

  3. path('todo_create',views.todo_create),

    The url is for creating new todo

  4. path('delete_todo/', views.delete_todo)

    The url is for deleting a particular todo

  5. path('update_todo/',views.update_todo)

    This is for updating todos with a particular id

  6. path('search', views.search)

    for searching for different types of todo

  7. path('users',UsersList.as_view())

    for displaying the list of avaliable users

  8. path('/create_comment',CreateComment.as_view())

    for creating comments to a particular todo
